## App Review Notes (for Apple reviewers)

```
Vizzy is a portfolio tracking app with AI-powered insights. The app uses RevenueCat for in-app purchases.

To test the app:
1. Open the app — you'll see the onboarding flow
2. Enter any name and answer the 5 interview questions
3. Navigate to the Vault tab to see portfolio categories
4. Use the + button to add holdings manually
5. Visit the Ask Vizzy tab to chat with the AI
6. Visit the Briefing tab for daily AI-generated cards
7. The paywall appears when free limits are reached (3 chat messages, 2 briefing cards)

No login is required. All data is stored locally on device.

API keys are bundled for demo purposes. No backend server is needed.
```
